Css function

$("#css").on("click", "li", function(){ $(this).css({"background-color": "#000000", "border-color": "#666666"}); })

It's better to use a Javascript object {""attr":"value", "attr":"value"} than to chain .css functions

Also it's always better to use add/remove/toggle class instead of writting css inside of the javascript

Class

$("#class").on("click", "li", function(){ $(this).toggleClass("color"); })

Animate

$("#animate").on("click", "li", function(){ $(this).animate({"margin-top" : "-10px"}); });

Animation speed

$("#speed").on("click", "li", function(){ $(this).animate({"margin-top" : "-10px"}, 'slow'); });

Css Animations

$("#css-animation").on("click", "li", function(){ $(this).toggleClass("animated-color"); });
